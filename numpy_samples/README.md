# NumPy Code Samples

A topic-by-topic set of NumPy code samples. Each file is runnable and focused on one knowledge area.

**How To Run**

```bash
python -m numpy_samples.samples.00_setup
python -m numpy_samples.samples.05_broadcasting
```

**Topics**

- 00_setup: environment and NumPy version
- 01_array_creation: array creation methods
- 02_indexing_slicing: basic indexing and slicing
- 03_boolean_fancy: boolean and fancy indexing
- 04_reshape_transpose: reshape, transpose, ravel
- 05_broadcasting: broadcasting rules
- 06_dtypes_casting: dtype inspection and casting
- 07_ufuncs: universal functions
- 08_aggregation_axis: aggregation across axes
- 09_stack_concat: stack/concat/split
- 10_sort_unique: sorting and unique
- 11_random: random sampling
- 12_linear_algebra: dot, matmul, solve
- 13_statistics: mean/var/std/percentile
- 14_masking_nan: mask, NaN handling
- 15_io_npy_csv: save/load npy and CSV
- 16_views_copies: views vs copies
- 17_performance_tips: vectorization basics
- 18_structured_arrays_optional: structured arrays
