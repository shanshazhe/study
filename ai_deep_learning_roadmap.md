# 深度学习与人工智能学习路线

> 适用人群：零基础到中级工程水平
> 目标：理解核心概念，能独立完成模型训练与小型项目落地

---

## 0. 准备阶段（1-2 周）

- **学习目标**：明确方向与学习方法，搭建开发环境。
- **内容**：
  - 认识 AI / ML / DL 的区别与应用场景
  - 学习路线与目标拆解
  - 安装开发环境：Python、Jupyter、CUDA（如需 GPU）

---

## 1. 数学与编程基础（4-6 周）

- **数学**
  - 线性代数：矩阵运算、特征值/特征向量、SVD
  - 概率与统计：分布、期望、方差、贝叶斯
  - 微积分：导数、链式法则、梯度

- **编程（Python）**
  - Python 基础语法与面向对象
  - Numpy / Pandas / Matplotlib
  - 代码组织与调试习惯

---

## 2. 机器学习基础（4-6 周）

- **核心概念**
  - 训练/验证/测试集
  - 过拟合与欠拟合
  - 损失函数、梯度下降

- **经典算法**
  - 线性回归、逻辑回归
  - 决策树、随机森林
  - 支持向量机（SVM）
  - 聚类（K-Means）

- **实践**
  - 使用 sklearn 做小数据集实验
  - 学会模型评估与调参

---

## 3. 深度学习核心（6-10 周）

- **神经网络基础**
  - 前向传播与反向传播
  - 激活函数（ReLU、Sigmoid、Tanh）
  - 优化方法（SGD、Adam）

- **常见网络结构**
  - MLP（全连接）
  - CNN（卷积网络）
  - RNN / LSTM / GRU

- **实践**
  - 使用 PyTorch 或 TensorFlow 完成图像/文本任务
  - 自己实现一个简单网络

---

## 4. 进阶方向（任选 1-2 个，6-12 周）

- **计算机视觉**
  - 目标检测、图像分割
  - ResNet / YOLO / U-Net

- **自然语言处理**
  - 词向量、Transformer
  - BERT、GPT 系列基础

- **强化学习**
  - 马尔可夫决策过程
  - Q-Learning、Policy Gradient

---

## 5. 工程化与项目实战（持续）

- 模型部署（REST API / Web / Edge）
- 模型压缩与推理优化
- 数据管道与 MLOps
- 参加竞赛或完成真实项目

---

## 6. 推荐实践节奏（示例）

- **每周**：
  - 3-4 天学习理论
  - 2-3 天代码实践
  - 1 天复盘总结

- **每阶段完成一个小项目**
  - 阶段 2：房价预测
  - 阶段 3：手写数字识别
  - 阶段 4：文本分类 / 图像检测

---

## 7. 小建议

- 理论与实践必须同步推进
- 做项目时要写总结与复盘
- 每个阶段输出成果：代码 + 笔记 + 复盘
