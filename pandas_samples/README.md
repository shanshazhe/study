# Pandas Code Samples

A pragmatic, topic-by-topic set of pandas code samples. Each file is runnable and focused on a single knowledge area.

**How To Run**

```bash
python -m pandas_samples.samples.00_setup
python -m pandas_samples.samples.01_series_dataframe
```

**Topics**

- 00_setup: environment and pandas version
- 01_series_dataframe: Series, DataFrame, basic creation
- 02_indexing_loc_iloc: loc/iloc, selection, slicing
- 03_filter_sort: boolean filtering, sorting, nlargest
- 04_missing_data: isna, fillna, dropna
- 05_dtypes_conversion: astype, to_numeric, to_datetime
- 06_string_ops: vectorized string methods
- 07_datetime_timeseries: dt accessors, resample
- 08_groupby_agg: groupby, agg, transform
- 09_merge_join_concat: merge, join, concat
- 10_reshape_pivot_melt: pivot, pivot_table, melt
- 11_multiindex: multi-level index basics
- 12_apply_map_vectorize: apply, map, vectorized ops
- 13_window_rolling_ewm: rolling windows, ewm
- 14_categorical: categorical dtype
- 15_io_csv_json: read/write CSV and JSON
- 16_io_excel_parquet_optional: optional IO engines
- 17_plotting_optional: plotting with matplotlib
- 18_duplicates_unique: duplicates, nunique
- 19_memory_optimization: memory usage and optimization
- 20_chunks_large_files: chunked reading
- 21_query_eval: query and eval
- 22_data_validation: basic validation patterns
- 23_style_optional: Styler basics for notebooks
- 24_groupby_advanced: advanced groupby patterns
- 25_timeseries_advanced: time series advanced patterns
- 26_performance_optimization: performance tips
- 27_string_processing_advanced: advanced string processing
- 28_merge_join_advanced: advanced merge/join
- 29_plotting_advanced_optional: advanced plotting with matplotlib
- 30_missing_data_advanced: advanced missing data handling

**Notes**

- The goal is broad coverage of common pandas knowledge points. If you want deeper coverage of any topic, say which ones and I will expand.
- Some samples use optional dependencies. Those files mention the needed package in comments.
