# Matplotlib Code Samples

A topic-by-topic set of Matplotlib code samples. Each file is runnable and focused on one knowledge area.

**How To Run**

```bash
python -m matplotlib_samples.samples.00_setup
python -m matplotlib_samples.samples.05_subplots
```

**Topics**

- 00_setup: environment and Matplotlib version
- 01_line_plot: basic line plot
- 02_scatter_plot: scatter plot with size and color
- 03_bar_plot: bar and horizontal bar
- 04_hist_plot: histogram and bins
- 05_subplots: multiple subplots
- 06_style_colors: styles and colors
- 07_annotations: annotate and text
- 08_grid_ticks: grid and tick formatting
- 09_dates_timeseries: date axis and formatting
- 10_images_heatmap: imshow and heatmap
- 11_twin_axes: secondary axis
- 12_savefig: save figures
- 13_colormaps: colormap examples
- 14_3d_optional: 3D plotting (optional)

**Notes**

- Most samples save output images to `/tmp`. If you are on Windows, change the output path.
- Some samples require optional dependencies (e.g., `mpl_toolkits` for 3D).
